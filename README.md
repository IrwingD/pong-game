# 🏓 Pong Game (React + TypeScript + Canvas)

A clean, engine-structured Pong implementation built with:

- ⚛️ React (Vite)
- 🔷 TypeScript
- 🎨 HTML5 Canvas
- 🚀 Designed for future multiplayer expansion

---

## 🎯 Project Goals

This project is structured like a small game engine instead of a single React component.  

The goal is to:

- Separate rendering from game logic
- Keep physics deterministic
- Make it easy to convert to multiplayer later
- Keep the React layer thin and focused only on UI
